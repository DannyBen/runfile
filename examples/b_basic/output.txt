Runner 0.1.0
The first ever Runfile program

Usage:
  run greet NAME [--long --color]
  run (-h|--help|--version)

Commands:
  greet NAME [--long --color]
      Say hello to NAME

Options:
  --long
      Show a longer greeting

  --color
      Use colored output

  -h --help
      Show this screen

  --version
      Show version number
