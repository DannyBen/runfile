Runfile

Usage:
  run watch css
  run watch js
  run watch a-movie
  run watch friends-again
  run git (commit|push|pull|branch|log)
  run (-h|--help)

Commands:
  watch css
      Watch files of type css

  watch js
      Watch files of type js

  watch a-movie
      Watch files of type a-movie

  watch friends-again
      Watch files of type friends-again

  git (commit|push|pull|branch|log)
      Execute a git command

Options:
  -h --help
      Show this screen
